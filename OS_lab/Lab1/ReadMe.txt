The objective of project is to show how to create process using fork() command in UNIX.

Source files:
part1a.c
part1b.c
part2.c
progb.c

Executable files:
part1a
part1b
part2
progb

Instructions:
Use the "make all" command to compile all source files

Execution:
Part1a:
$:./part1a

$:./part1b

$:./part2 n     //where n is the integer the user wants to enter
